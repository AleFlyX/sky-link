1. \u0000-\u001f（C0 控制字符）

这是 ASCII 码表中最前面的 32 个字符（0–31），包括：

不可见字符：空字符 \0、制表符 \t、换行 \n、回车 \r、退格 \b 等。

通信控制字符：如 ACK（确认）、NAK（否定应答）、ESC（转义）等。

注意：\t、\n、\r 虽常被视为空白符，但在严格定义中属于控制字符，所以也会被匹配。

2. \u007f-\u009f（DEL + C1 控制字符）

\u007f：即 DEL（删除符），ASCII 的第 127 个字符。

\u0080-\u009f：即 C1 控制字符，来自 Latin-1 扩展（ISO-8859-1）中 128–159 的字符，常用于旧式终端控制（如 IND、NEL 等），在现代文本中很少出现，但某些编码转换时可能残留。
